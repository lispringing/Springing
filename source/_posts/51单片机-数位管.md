# 数码管的分类
数码管在51单片机上就是可以显示1234567890.的那个
一共是8个led组成，按照编号是abcdef(这些是数字管)，还有一个dp(这个是小数点管)
一般来说数码管分两类，分别是共阳极数码管和共阴极数码管
共阳极数码管:
所有led灯的阳极全接在一起，然后接入vcc 引脚给低电平就亮
一般说单片机用这种接法，因为51上电默认P1/P2/P3 上电高P0 上电高阻浮空，这样上电不烧程序数码管默认不会亮。

共阴极数码管:
所有 LED 负极接一起接地，之后给高电平就亮

可以看到这个数码管是共阳的数码管(因为四个管的引脚都接入了vcc上)
而且引出了两个端口

| 端口 | 作用说明 |
| ---- | -------- |
| P0   | 段选脚，控制数码管a~g、dp亮灭，决定显示数字 |
| P2   | 位选脚，选择点亮4位中指定哪一个数码管 |

>这里要注意一个问题就是这四个数码管是没法同时点亮的，但是我们可以这样；让1号亮1ms，然后2号亮1ms，3号亮1ms，依次循环，只要时间够短，就看不出来

# 位选码

| 端口 | 作用说明 |
| ---- | -------- |
| P2=0xFE   | 第一位亮 |
| P2=0xFD   | 第二位亮 |
| P2=0xFB   | 第三位亮 |
| P2=0xF7   | 第四位亮 |
| P2=0xFF   | 全部熄灭 |

# 让数码管显示1111

```c
#include <REGX52.H>
#include <INTRINS.H>

// 共阳数码管段码表：1 对应的段码是 0xF9
unsigned char code SEGMENT_1 = 0xF9;

void Delay500ms()		//@11.0592MHz
{
	unsigned char i, j, k;

	_nop_();
	_nop_();
	i = 22;
	j = 3;
	k = 227;
	do
	{
		do
		{
			while (--k);
		} while (--j);
	} while (--i);
}

// 1ms 延时，用于数码管动态扫描
void Delay1ms()		//@11.0592MHz
{
	unsigned char i, j;

	i = 2;
	j = 239;
	do
	{
		while (--j);
	} while (--i);
}

void main()
{
	while(1)
	{
		// 第1位显示1
		P2 = 0xFE;   // 选中第1位
		P0 = SEGMENT_1;
		Delay1ms();
		
		// 第2位显示1
		P2 = 0xFD;   // 选中第2位
		P0 = SEGMENT_1;
		Delay1ms();
		
		// 第3位显示1
		P2 = 0xFB;   // 选中第3位
		P0 = SEGMENT_1;
		Delay1ms();
		
		// 第4位显示1
		P2 = 0xF7;   // 选中第4位
		P0 = SEGMENT_1;
		Delay1ms();
	}
}
```
