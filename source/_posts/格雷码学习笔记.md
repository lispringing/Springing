# 普通二进制
我们平时的二进制是这样（2 位）：
0 → 00
1 → 01
2 → 10
3 → 11
你看 1 变成 2：(十进制)
01 → 10
两个位同时从 0 变 1、1 变 0
机器在快速切换时，很容易读一半、读错。
比如本来要变 10，结果机器读到 00 或 11，就出错了。

# 格雷码
格雷码规定：
从一个数变到下一个数，只允许变一位 0/1
2 位格雷码：
0 → 00
1 → 01
2 → 11
3 → 10
你看变化：
0→1：只变最后一位
1→2：只变第一位
2→3：只变最后一位
永远只变一位
机器就不会读错。
3 位格雷码更明显
十进制 → 格雷码：
0 → 000
1 → 001
2 → 011
3 → 010
4 → 110
5 → 111
6 → 101
7 → 100
每相邻两个，永远只变一位。
# 用来干嘛
比如电机旋转编码器：
电机转一点点，数字就变一下。
如果用普通二进制，可能同时变多位，容易读错位置。
用格雷码，只变一位，绝对不会读错。
# 总结
格雷码 = 一种特殊二进制，相邻数字只变一位，防止机器读数出错。
